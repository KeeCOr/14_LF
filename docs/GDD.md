# LotteryFantasy — 게임 기획서

> 최종 업데이트: 2026-05-06

---

## 1. 게임 개요

| 항목 | 내용 |
|---|---|
| 장르 | 룰렛 × 타워 디펜스 × 카드 게임 |
| 플랫폼 | PC / 모바일 (Unity 2022 URP 2D) |
| 인원 | 1인 vs AI / 생존 모드 (싱글) |
| 한 판 시간 | 전투: 3분 / 생존: 무제한 |
| 수익화 | F2P + IAP (캐릭터 획득·성장 중심) |

### 핵심 루프

```
룰렛 자동 회전
  → 행운 소모 (릴 1개 정지당 1회)
  → 3릴 모두 정지 → 속성 에너지(불/철/생) 획득
  → 에너지 소모 → 카드 배치 (유닛 / 건물)
  → 유닛이 몬스터 격퇴 → 기지 방어
  → 적 기지 파괴 → 승리
```

## 2. 게임 모드

| 모드 | 설명 |
|---|---|
| **전투 모드** | 3분 제한 시간 내 AI 기지 파괴. STAGE 시스템으로 난이도 상승 |
| **생존 모드** | 무한 웨이브를 최대한 버팀. 웨이브마다 몬스터 수·엘리트 비율 증가 |

- 게임 시작 시 덱에서 **랜덤 카드 1장** 손패에 지급
- 승패 확정 시 `Time.timeScale = 0`으로 즉시 게임 정지
- 결과 화면에서 재시작 버튼 → timeScale 복원 후 씬 리로드

## 4. 속성 에너지 시스템

룰렛 결과로 획득되는 자원. 카드 배치 시 소모된다.

| 속성 | 아이콘 | UI 색상 |
|---|---|---|
| 불(Fire) | 🔥 | #FF8833 |
| 철(Iron) | ⚔ | #99CCFF |
| 생(Life) | 💚 | #33FF77 |

- HUD **우상단**에 아이콘 + 숫자 표시. 0일 때 회색으로 흐려짐
- 배치 실패(에너지 부족) 시 부족한 속성 레이블 빨간색 점멸(4회)
- 손패 카드 비용: 속성 색상으로 표시, 부족한 속성은 빨간색 강조

## 5. 룰렛 시스템

### 5.1 행운(Luck) 충전

| 항목 | 수치 |
|---|---|
| 행운 자동 충전 주기 | 2초마다 +1 |
| 시작 시 지급 | 행운 3회 |
| 릴 1개 정지 비용 | 행운 1회 |

### 5.2 릴 동작 방식

- 룰렛은 **항상 자동으로 회전** (게임 내내 멈추지 않음)
- 버튼 클릭마다 **행운 1 소모** → 릴 하나 정지
- 3개 모두 정지 → 에너지 계산 후 자동으로 다음 스핀 시작

### 5.3 에너지 계산 규칙

| 결과 | 조건 | 에너지 보상 |
|---|---|---|
| 올미스 | 3릴 모두 다른 속성 | 각 속성 +1 (총 3) |
| 더블 | 2릴 동일 속성 | 해당 +2, 나머지 +1 (보너스 +1) |
| 트리플 | 3릴 모두 동일 | 해당 속성 +6 (보너스 +3) |

> 트리플 달성 시 금색 화면 플래시 연출 발생

## 11. 유닛 AI

```
매 프레임:
  1. sightRange 이내 가장 가까운 몬스터 탐색 (대공 필터 적용)
  2. 몬스터 발견 → 추격 + attackRange 내 공격
  3. 몬스터 없음 + 포탈 사거리 내 → 포탈 공격 (플레이어 유닛만)
  4. 위 모두 해당 없음 → 전방 전진
```

- 타겟 탐색: `MonsterController.AllMonsters` / `UnitController.ActivePlayerUnits` 정적 리스트 활용

## 13. 포탈 시스템

| 속성 | 수치 |
|---|---|
| HP | 500 |
| 엘리트 소환 임계값 | HP 40% 미만 |
| 색상 | HP 100%=보라, HP 0%=빨강 |

| 조건 | 효과 |
|---|---|
| 피격마다 | 포탈 우측(x=+1)에서 일반 몬스터 소환 |
| HP < 40% + 피격 | 추가 엘리트 몬스터 소환 |

## 15. 스테이지 시스템 (전투 모드)

| 스테이지 | 경과 시간 | 웨이브 간격 | 엘리트 출현 |
|---|---|---|---|
| STAGE 1 | 0 ~ 60초 | 5초 | 없음 |
| STAGE 2 | 60 ~ 120초 | 4.5초 | 1/3 확률 |
| STAGE 3 | 120초 ~ | 4초 | 1/2 확률 |

---

## 16. 생존 모드 웨이브

| 항목 | 공식 |
|---|---|
| 웨이브당 몬스터 수 | min(1 + (웨이브-1)/3, 5마리) |
| 엘리트 출현 확률 | min((웨이브-1) × 7%, 60%) |
| 웨이브 간격 | max(2초, 6초 - 웨이브 × 0.18초) |
