# LotteryFantasy — 게임 기획서 (MVP)

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 슬롯머신 × 타워 디펜스 |
| 플랫폼 | PC (Unity 2022 URP 2D) |
| 인원 | 1인 (vs AI) |
| 한 판 시간 | 3분 |

**핵심 루프**  
몬스터 처치 → XP 획득 → SPIN → 카드(유닛) 획득 → 유닛 배치 → 기지 방어

---

## 2. 씬 레이아웃

```
PlayerVillage(-7.5)  PlayerSpawn(-2.5)  [포탈](0)  EnemySpawn(2.5)  EnemyVillage(7.5)
     기지(시안)          몬스터 출구          중앙              몬스터 출구        기지(오렌지)
```

- **좌측 영역 (x < 0)**: 플레이어 진영. 몬스터가 PlayerSpawn에서 출현해 기지로 이동.
- **우측 영역 (x > 0)**: AI 진영. 동일한 구조.
- **포탈 (x = 0)**: 플레이어 유닛이 공격할 수 있는 중앙 구조물.

---

## 3. 핵심 시스템

### 3.1 슬롯 머신

| 단계 | 설명 |
|------|------|
| XP 획득 | 내 진영 몬스터 처치 시 xpReward(50) XP 적립 |
| 스핀 충전 | XP 100 모일 때마다 스핀 1회 충전 |
| SPIN | 덱(12장: 검사×4, 궁수×4, 기사×4)에서 3장 무작위 뽑기 |
| 결과 판정 | Triple → 강화 카드 획득 / Double → 일반 카드 획득 / AllDifferent → 전체 버프 |

- 릴 3개에 뽑힌 카드 이름이 표시됨
- 매치 확률 약 78% (AllDifferent 22%)

### 3.2 카드 & 핸드

- 핸드 슬롯 4개
- 카드 클릭 → 선택(초록 하이라이트) → 화면 왼쪽 클릭 → 유닛 배치
- 배치된 유닛은 플레이어 진영(x ≤ 0)에서만 이동 가능

### 3.3 유닛

| 유닛 | HP | 공격력 | 속도 | 사거리 | 공격속도 |
|------|----|--------|------|--------|---------|
| 검사 | 80 | 15 | 2.0 | 1.5 | 1/초 |
| 궁수 | 50 | 10 | 1.5 | 5.0 | 2/초 |
| 기사 | 120 | 20 | 1.2 | 1.0 | 0.8/초 |

- 가장 가까운 몬스터를 우선 공격
- 몬스터가 없으면 포탈(x=0)을 향해 전진 후 공격

### 3.4 포탈 시스템

| 조건 | 효과 |
|------|------|
| 피격마다 | 플레이어 진영에 일반 몬스터 1마리 즉시 소환 |
| HP 40% 미만 피격 | 정예 몬스터 추가 소환 (HP 150, 공격력 20, 속도 1.8) |

- 포탈 HP 500, 색상으로 체력 표시 (보라 → 빨강)
- 포탈 공격은 기지를 빠르게 방어하는 대신 몬스터 압박을 높이는 고위험-고보상 선택

### 3.5 몬스터 전달 (TransferSystem)

- 플레이어 진영 몬스터 처치 → 1.5초 후 동일 몬스터가 적 진영에 소환
- 적 진영 몬스터 처치 → 1.5초 후 플레이어 진영에 소환
- 상대방에게 압박을 전가하는 메커니즘

### 3.6 AI 상대방

- 스핀 간격: 4초 / 유닛 배치 간격: 6초
- 게임 시작 시 3회 스핀 선충전 (240 XP)
- 전략: 무작위 위치에 유닛 배치

## 4. 승패 조건

| 조건 | 결과 |
|------|------|
| 내 기지 HP 0 | 패배 |
| 적 기지 HP 0 | 승리 |
| 3분 타임오버 | 기지 HP 높은 쪽 승리 |
