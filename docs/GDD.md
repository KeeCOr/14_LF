# LotteryFantasy — 게임 기획서

> 최종 업데이트: 2026-04-29

---

## 1. 게임 개요

| 항목 | 내용 |
|---|---|
| 장르 | 슬롯머신 × 타워 디펜스 × 카드 게임 |
| 플랫폼 | PC / 모바일 (Unity 2022 URP 2D) |
| 인원 | 1인 vs AI / 생존 모드 (싱글) |
| 한 판 시간 | 전투: 3분 / 생존: 무제한 |
| 수익화 | F2P + IAP (캐릭터 획득·성장 중심) |

### 핵심 루프

```
몬스터 처치 → XP → 스핀 충전 → SPIN → 카드 획득
                                              ↓
                               유닛카드: 스핀 소모 → 유닛 배치 → 기지 방어
                               스킬카드: 지점 선택 → 즉발 범위 피해
                               버프카드: 클릭 즉시 발동 → 아군 강화
```

## 2. 게임 모드 선택 (시작 메뉴)

게임 시작 시 모드 선택 화면이 표시된다.

| 모드 | 설명 |
|---|---|
| **전투 모드** | 3분 제한 시간 내 AI 기지 파괴. STAGE 시스템으로 난이도 상승 |
| **생존 모드** | 무한 웨이브를 최대한 버팀. 웨이브마다 몬스터 수·엘리트 비율 증가 |

- 게임 시작 시 덱에서 **랜덤 카드 1장** 손패에 지급

## 3. 씬 레이아웃

```
x=-7.5        x=-2.5      x=0        x=+2.5       x=+7.5
[플레이어기지]  [적웨이브출구]  [포탈]  [AI웨이브출구]  [AI기지]
  (시안)                  (보라/빨강)                (오렌지)
```

| 영역 | 설명 |
|---|---|
| 좌측 (x < 0) | 플레이어 진영. 몬스터가 x=-2.5에서 출현해 기지(-7.5)로 이동 |
| 우측 (x > 0) | AI 진영. 동일 구조 |
| 중앙 (x = 0) | 포탈. 플레이어 유닛의 공격 목표이자 적 증원의 출구 |

## 4. 슬롯머신 시스템

### 4.1 XP & 스핀 충전

| 항목 | 수치 |
|---|---|
| 처치 XP | 일반 몬스터 50, 엘리트 150 |
| 스핀 1회 충전에 필요한 XP | 100 |
| 게임 시작 지급 | 스핀 3회 (XP 300) + 랜덤 카드 1장 |

> 스핀 충전량은 유닛 배치 비용으로도 소모됨. 충전량이 부족하면 스핀도 배치도 불가.

### 4.2 덱 구성 (18장 / 7종)

| 카드 | 종류 | HP | 피해 | 이동속도 | 사거리 | 공격속도 | 시야 | 배치비용 | 장수 |
|---|---|---|---|---|---|---|---|---|---|
| 검사 | 유닛 | 80 | 15 | 2.0 | 1.5 | 1.0/s | 5 | 스핀 1 | 3 |
| 궁수 | 유닛 | 50 | 10 | 1.5 | 5.0 | 2.0/s | 8 | 스핀 1 | 3 |
| 기사 | 유닛 | 120 | 20 | 1.2 | 1.0 | 0.8/s | 4 | 스핀 2 | 3 |
| 마법사 | 유닛 | 40 | 28 | 1.8 | 4.5 | 0.6/s | 8 | 스핀 2 | 3 |
| 힐러 | 유닛 | 70 | 0 | 1.6 | 2.0 | 0.8/s | 6 | 스핀 2 | 2 |
| 번개화살 | 스킬 | — | 80 (범위) | — | 반경 2.0 | — | — | 무료 | 2 |
| 포탈폭격 | 스킬 | — | 120 (범위) | — | 반경 3.0 | — | — | 무료 | 2 |

> **시야(sightRange)**: 이 거리 이내에 몬스터가 있을 때만 추격. 없으면 전방 전진.
> **힐러**: 피해 대신 주변 아군에게 초당 heal=8 회복 적용.

### 4.3 릴 판정

릴 3개를 덱에서 **복원 추출**. 카드 인스턴스 동일성(`==`)으로 판정.

| 결과 | 조건 | 확률 | 보상 |
|---|---|---|---|
| Triple | 3릴 모두 동일 카드 | 약 6% | 해당 카드 손패 추가 (향후: 스탯 강화) |
| Double | 2릴 동일 카드 | 약 56% | 해당 카드 손패 추가 |
| AllDifferent | 3릴 전부 다름 | 약 37% | 버프카드 손패 추가 |

> **Triple 현황**: 현재 구현에서 Triple로 획득한 카드는 스탯 강화 없이 동일 카드를 손패에 추가함.
> `CardTier.Enhanced` 구조는 준비되어 있으나 실제 강화 효과는 미구현 (향후 과제 참조).

## 5. 카드 & 손패

- 손패 슬롯 4개. 가득 찬 경우 새 카드 획득 불가.
- 카드 종류별 UI 색상으로 구분:

| 카드 종류 | 버튼 색상 | 사용 방법 |
|---|---|---|
| 유닛카드 (미선택) | 어두운 파랑 | 클릭 선택 → 배치 구역 클릭 **또는 드래그** → 배치 |
| 유닛카드 (선택됨) | 초록 | — |
| 스킬카드 (미선택) | 어두운 보라 | 클릭 선택 → 원하는 지점 클릭 → 즉시 발동 |
| 스킬카드 (선택됨) | 밝은 보라 | — |
| 버프카드 | 금색 | 클릭 즉시 발동 (`[클릭 → 즉시 발동]` 텍스트 표시) |

### 유닛 배치 규칙

- 배치 위치: 플레이어 진영 (x ≤ -0.5) 에서만 가능
- 배치 방법: 카드 클릭 선택 후 배치 구역 클릭 **또는** 카드를 배치 구역으로 직접 드래그
- 배치 비용: 스핀 충전량 소모 (카드마다 다름: 검사/궁수 1스핀, 기사/마법사/힐러 2스핀)
- 충전량 부족 시 배치 취소
- 배치 구역 오버레이: 유닛카드 선택 시 파란색 강조 표시

## 6. 스킬 카드 시스템

스킬카드는 선택 후 화면 지점을 클릭하면 해당 위치 반경 내 모든 적에게 즉시 피해.

| 스킬 | 피해 | 범위 반경 | 비용 | 발동 이펙트 |
|---|---|---|---|---|
| 번개화살 | 80 | 2.0 | 무료 | 보라색 화면 플래시 |
| 포탈폭격 | 120 | 3.0 | 무료 | 보라색 화면 플래시 |

> 스킬카드는 스핀 비용 없이 사용 가능. SPIN 결과가 아닌 특별 획득 경로.

## 7. 유닛 AI

모든 유닛(플레이어 & AI)은 동일한 우선순위 로직으로 동작.

```
매 프레임:
  1. 시야(sightRange) 이내에서 가장 가까운 몬스터 탐색
  2. 몬스터 발견 → 추격 + 사정거리(attackRange) 내 공격
  3. 몬스터 없음 + 포탈 사정거리 내 → 포탈 공격 (플레이어 유닛만)
  4. 위 모두 해당 없음 → 전방 전진
```

| 유닛 종류 | 전진 방향 | 경계선 |
|---|---|---|
| 플레이어 유닛 | 오른쪽(+x) | x = 0 (넘지 않음) |
| AI 유닛 | 왼쪽(-x) | x = 0 (넘지 않음) |

- 물리 기반 이동: `Rigidbody2D` + `CircleCollider2D` → 자연스러운 군집, 겹침 방지

## 8. 버프 시스템

### 버프카드 획득

- AllDifferent 스핀 결과 시 "전투 버프" 카드가 손패에 추가됨
- 즉시 발동이 아닌 **카드 형태**로 보관 → 원하는 타이밍에 사용 가능

### 버프 효과

| 버프 종류 | 공격 배율 | 이동속도 배율 | 지속 시간 |
|---|---|---|---|
| 공격 버프 | ×2.0 | ×1.0 | 8초 |
| 속도 버프 | ×1.0 | ×1.8 | 12초 |
| 균형 버프 | ×1.5 | ×1.2 | 10초 |

- 버프는 **모든 플레이어 유닛**의 공격/이동에 전역(static) 배율로 적용
- 지속 시간 후 자동 해제, 중첩 적용 가능 (배율 누적)
- 배치 스핀 비용: 0 (무료)

## 9. 포탈 시스템

| 속성 | 수치 |
|---|---|
| HP | 500 |
| 엘리트 소환 임계값 | HP 40% 미만 |
| 색상 | HP 100% = 보라, HP 0% = 빨강 (Lerp 그라디언트) |

### 피격 반응

| 조건 | 효과 |
|---|---|
| 피격마다 | 포탈 우측(x=+1)에서 일반 몬스터 즉시 소환 |
| HP < 40% + 피격 | 포탈 우측에서 엘리트 몬스터 추가 소환 |

> 포탈을 빠르게 공략할수록 더 강한 적이 몰려오는 **고위험-고보상** 구조

## 10. 증원 시스템 (TransferSystem)

- **트리거**: 몬스터 처치
- **지연**: 1.5초
- **결과**: 처치된 진영 포탈 우측(x=+1)에서 동일 등급 몬스터 재소환

```
플레이어가 몬스터 처치
  → 1.5초 후 포탈에서 새 몬스터 출현 (같은 플레이어 진영으로)
```

> 더 많이 처치할수록 포탈에서 적이 계속 나온다 — 방어 효율 vs 압박 증가의 딜레마

## 11. 스테이지 시스템 (전투 모드)

경과 시간에 따라 STAGE가 상승하여 웨이브 난이도가 증가한다.

| 스테이지 | 경과 시간 | 웨이브 간격 | 엘리트 출현 |
|---|---|---|---|
| STAGE 1 | 0 ~ 60초 | 5초 | 없음 |
| STAGE 2 | 60 ~ 120초 | 4.5초 | 1/3 확률 포함 |
| STAGE 3 | 120초 ~ | 4초 | 1/2 확률 포함 |

- HUD 상단에 현재 스테이지 표시 (`STAGE 1` / `STAGE 2` / `STAGE 3`)

## 20. 성공 가능성 포인트

### 20.1 핵심 차별화 요소

**슬롯머신 × 타워디펜스 × 카드게임의 3중 장르 퓨전**

- 슬롯머신의 불확실성 → 카드 구성의 변동성 → 상황 대응 전략
- 타워디펜스의 배치 판단 → 포탈 공략 딜레마 → 자원(스핀) 관리
- 카드게임의 손패 관리 → 타이밍 플레이 (버프카드 보관) → 심리전

### 20.2 독창적인 게임플레이 딜레마

| 딜레마 | 내용 |
|---|---|
| **포탈 공략 딜레마** | 빠르게 포탈을 공략하면 엘리트 몬스터가 쏟아짐. 천천히 하면 적 기지를 못 뚫음 |
| **증원 딜레마** | 몬스터를 많이 처치할수록 증원 스폰이 증가. 방어 효율과 압박 증가가 상충 |
| **스핀 자원 딜레마** | 스핀을 카드 뽑기에 쓸지, 강한 유닛 배치 비용으로 쓸지 매번 선택 필요 |

### 20.3 도박 심리 + 전략의 시너지

- Triple(6%)의 희귀도가 짜릿함을 만듦
- AllDifferent(37%)의 버프카드가 역전 기회를 제공
- 결과 예측 불가 → 매 판 다른 덱 구성 → 재플레이 동기

### 20.4 3분 단판 구조의 중독성

- 짧은 플레이 세션 → "한 판만 더" 심리 극대화
- 패배해도 즉시 재도전 가능 → 이탈률 감소
- 모바일 이식 시 짧은 세션 타임이 강점으로 작용

### 20.5 확장 설계 잠재력

- `CardTier.Enhanced` 구조가 준비됨 → Triple 강화 효과 구현 가능 + 별 등급 돌파 시스템 연동 기반
- 덱 편집 추가 시 사전 전략 레이어 부여 → 하드코어 플레이어층 확보
- PvP 네트워크 대전 → 대칭 구조(좌우 진영)가 이미 PvP 설계에 최적화됨
- 캐릭터 성장 시스템(레벨·별 등급·장비) → F2P BM의 핵심 지출 동선 (§21 참조)

### 20.6 시장 접근성

| 항목 | 평가 |
|---|---|
| 진입 장벽 | 낮음. 규칙이 직관적이며 첫 판부터 플레이 가능 |
| 모바일 이식 잠재력 | 높음. 탭/드래그 기반 조작, 짧은 세션 |
| F2P 수익화 적합도 | 높음. 캐릭터 수집·성장 BM, 코스메틱, 시즌 패스 등 (§21 참조) |
| 스트리머/유튜버 친화도 | 높음. 슬롯 연속 Triple, 역전 버프 등 클립성 순간이 자연 발생 |

## 21. 비즈니스 모델 (BM)

### 21.1 수익화 개요

| 항목 | 내용 |
|---|---|
| 모델 | F2P (무료 플레이) + IAP |
| 핵심 지출 동기 | 강한 캐릭터를 더 빨리 얻고, 더 강하게 키우고 싶다 |
| 결제 없이도 | 모든 콘텐츠 클리어 가능 (Pay-to-Win 배제) |
| 결제하면 | 성장 속도 가속 + 희귀 캐릭터 접근성 향상 |

> 슬롯머신 스핀 = 기존 게임플레이 루프 자체가 "뽑기" 심리와 동일 구조. BM이 게임 메커닉과 자연스럽게 연결됨.

---

### 21.2 재화 시스템

| 재화 | 획득 방법 | 용도 |
|---|---|---|
| **골드** | 전투 보상, 무료 일일 지급 | 유닛 강화 재료, 덱 슬롯 해금 |
| **크리스탈** (프리미엄) | IAP, 업적 보상 (소량) | 프리미엄 스핀, 캐릭터 즉시 해금, 슬롯 확장 |
| **스핀 티켓** | 무료 플레이, 일일 퀘스트 | 전투 중 추가 스핀 |

---

### 21.3 캐릭터 획득 시스템

#### 기본 풀 (무료)
- 현재 7종 유닛 (검사·궁수·기사·마법사·힐러·번개화살·포탈폭격)은 기본 풀로 무료 지급
- 전투 플레이만으로 슬롯 스핀을 통해 획득 가능

#### 프리미엄 풀 (크리스탈 소모)

| 등급 | 확률 | 특징 |
|---|---|---|
| 일반 (★★) | 60% | 기본 풀 유닛 |
| 희귀 (★★★) | 30% | 강화 스탯 + 패시브 1개 |
| 영웅 (★★★★) | 9% | 고유 스킬 + 강화 스탯 |
| 전설 (★★★★★) | 1% | 전용 스킨 + 최상위 스탯 |

- **천장 시스템**: 90회 이내 전설 미획득 시 다음 1회에서 전설 확정
- **픽업 배너**: 기간 한정 특정 전설 유닛 출현율 2배

---

### 21.4 캐릭터 성장 시스템

#### 레벨업
- 유닛마다 레벨 1~30 존재
- 골드 + 동일 유닛 카드 조각(Shard) 소모 → 레벨업

| 레벨 구간 | HP 배율 | 피해 배율 |
|---|---|---|
| 1 | ×1.0 | ×1.0 |
| 10 | ×1.3 | ×1.3 |
| 20 | ×1.7 | ×1.7 |
| 30 | ×2.2 | ×2.2 |

#### 별(Star) 등급 돌파

| 돌파 | 필요 조각 | 효과 |
|---|---|---|
| ★→★★ | 10장 | HP +10% |
| ★★→★★★ | 20장 | 피해 +10% |
| ★★★→★★★★ | 40장 | 고유 패시브 해금 |
| ★★★★→★★★★★ | 80장 | 스킬 강화 |
| ★★★★★→★★★★★★ | 150장 | 전설 폼 변환 (스킨 변경) |
