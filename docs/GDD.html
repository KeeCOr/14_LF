4.3 릴 판정

릴 3개를 덱에서 복원 추출. 카드 인스턴스 동일성으로 판정.

결과조건확률보상
Triple3릴 모두 동일 카드약 6%해당 카드 손패 추가 (향후: 스탯 강화)
Double2릴 동일 카드약 56%해당 카드 손패 추가
AllDifferent3릴 전부 다름약 37%버프카드 손패 추가
Triple 현황: 현재 구현에서 Triple은 스탯 강화 없이 동일 카드를 손패에 추가함. CardTier.Enhanced 구조는 준비되어 있으나 실제 강화 효과는 미구현.

5 카드 & 손패

손패 슬롯 4개. 가득 찬 경우 새 카드 획득 불가.

카드 종류버튼 색상사용 방법
유닛카드 (미선택)어두운 파랑클릭 선택 → 배치 구역 클릭 또는 드래그 → 배치
유닛카드 (선택됨)초록—
스킬카드 (미선택)어두운 보라클릭 선택 → 원하는 지점 클릭 → 즉시 발동
스킬카드 (선택됨)밝은 보라—
버프카드금색클릭 즉시 발동

유닛 배치 규칙

  • 배치 위치: 플레이어 진영 (x ≤ −0.5) 에서만 가능
  • 배치 방법: 카드 클릭 선택 후 배치 구역 클릭 또는 카드를 배치 구역으로 드래그
  • 배치 비용: 검사/궁수 1스핀, 기사/마법사/힐러 2스핀
  • 충전량 부족 시 배치 취소
  • 배치 구역 오버레이: 유닛카드 선택 시 파란색으로 강조 표시

6 스킬 카드 시스템

스킬카드는 선택 후 화면 지점을 클릭하면 해당 위치 반경 내 모든 적에게 즉시 피해.

스킬피해범위 반경비용발동 이펙트
번개화살802.0무료보라색 화면 플래시
포탈폭격1203.0무료보라색 화면 플래시
스킬카드는 스핀 비용 없이 사용 가능. 발동 시 ScreenFlash 보라색 이펙트 재생.

7 유닛 AI

매 프레임:
  1. 시야(sightRange) 이내에서 가장 가까운 몬스터 탐색
  2. 몬스터 발견  → 추격 + 사정거리(attackRange) 내 공격
  3. 몬스터 없음 + 포탈 사정거리 내 → 포탈 공격 (플레이어 유닛만)
  4. 위 모두 해당 없음 → 전방 전진
유닛 종류전진 방향경계선
플레이어 유닛오른쪽 (+x)x = 0 (넘지 않음)
AI 유닛왼쪽 (−x)x = 0 (넘지 않음)
물리 기반 이동: Rigidbody2D + CircleCollider2D → 자연스러운 군집, 겹침 방지

8 버프 시스템

AllDifferent 스핀 결과 시 버프카드가 손패에 추가됨. 카드 형태로 보관 → 원하는 타이밍에 사용.
버프 종류공격 배율이동속도 배율지속 시간
공격 버프×2.0×1.08초
속도 버프×1.0×1.812초
균형 버프×1.5×1.210초

버프는 모든 플레이어 유닛에 전역 배율로 적용. 지속 시간 후 자동 해제, 중첩 적용 가능.

20 성공 가능성 포인트

20.1 핵심 차별화 — 3중 장르 퓨전

  • 슬롯머신의 불확실성 → 카드 구성의 변동성 → 상황 대응 전략
  • 타워디펜스의 배치 판단 → 포탈 공략 딜레마 → 자원(스핀) 관리
  • 카드게임의 손패 관리 → 타이밍 플레이 → 심리전

20.2 구조적 딜레마 3가지

딜레마내용
포탈 공략 딜레마빠르게 공략하면 엘리트 몬스터 폭증. 천천히 하면 적 기지를 못 뚫음
증원 딜레마몬스터를 많이 처치할수록 증원 스폰 증가. 방어 효율과 압박 증가 상충
스핀 자원 딜레마스핀을 카드 뽑기에 쓸지, 강한 유닛 배치에 쓸지 매번 선택

20.3 시장 접근성

항목평가
진입 장벽낮음. 규칙이 직관적이며 첫 판부터 플레이 가능
모바일 이식 잠재력높음. 탭/드래그 기반 조작, 짧은 세션
F2P 수익화 적합도높음. 캐릭터 수집·성장 BM, 코스메틱, 시즌 패스
스트리머 친화도높음. 슬롯 Triple, 역전 버프 등 클립성 순간 자연 발생
