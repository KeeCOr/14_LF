핵심 루프

룰렛 자동 회전 (수동 or AUTO 모드) → 행운 소모 (릴 1개 정지당 1회) → 3릴 모두 정지 → 속성 에너지 (🔥불 / ⚔철 / 💚생) 획득 → 에너지 소모 → 카드 배치 (유닛 / 건물 / 스킬 / 버프) → 유닛이 몬스터 격퇴 → 기지 방어 → 적 기지 파괴 → 승리

5.1 행운(Luck) 충전

항목수치

최대 보유10회

5.2 릴 동작 방식

🔥
릴 1
⚔
릴 2
💚
릴 3
  • 룰렛은 항상 자동 회전 (게임 내내 멈추지 않음)
  • 정지 버튼 클릭마다 행운 1 소모 → 릴 하나 정지
  • 3개 모두 정지 → 에너지 계산 후 자동으로 다음 스핀 시작
  • AUTO 버튼: 켜면 행운 보유 시 자동으로 릴 정지 (0.28초 간격)

5.3 릴 연출

  • 회전 중: 심볼 0.04초 간격 교체 + 배경 파란빛 pulse
  • 정지 시: 슬로우다운 플래시(5프레임) → 최종 심볼 → 스케일 1.4× 바운스 → 속성 컬러 플래시 → 페이드
  • 정지 버튼: 사용 가능 시 초록 pulse, 불가능 시 어둡게
  • 행운 게이지: 버튼 아래 황금색 수평 바 + ★N / 10 (Xs)

5.4 에너지 계산 규칙

결과조건에너지 보상
올미스3릴 모두 다른 속성각 속성 +1 (총 3)
더블2릴 동일 속성해당 +2, 나머지 +1 (보너스 +1)
트리플3릴 모두 동일해당 속성 +6 (보너스 +3) + 금색 화면 플래시

11 유닛 AI

매 프레임 우선순위: 1. sightRange 내 적 몬스터 탐색 (대공 필터 적용) 2. 몬스터 발견 → 추격 + attackRange 내 공격 3. 힐러: 체력 낮은 아군 탐색 → 근접 힐 4. 적 유닛(AI 유닛) 탐색 → 공격 5. 적 없음 + 포탈 사거리 내 → 포탈 공격 (플레이어 유닛만) 6. 위 모두 해당 없음 → 전방 전진
  • 분리 조향: 모든 유닛 간 겹침 방지 (반경 0.65, 힘 2.0)
  • 스티키 타겟팅: 타겟 사망 전까지 유지 → 사망 시 재탐색
  • 플레이어↔AI 유닛 서로를 공격 대상으로 지정 가능
  • 전역 배율: GlobalDamageMultiplier, GlobalSpeedMultiplier (버프 카드 연동)
